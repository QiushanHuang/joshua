# 智能资产记账系统 (Asset Tracker)

一个功能丰富的个人资产管理和记账可视化系统，支持多币种、多层级分类管理、趋势分析和自动记账等功能。

## 🚀 核心功能

### 💰 资产管理
- **多币种支持**: 支持人民币(CNY)、新加坡元(SGD)、美元(USD)、马来西亚令吉(MYR)
- **实时汇率转换**: 自动转换不同币种，支持汇率自定义设置
- **多层级分类**: 无限层级的资产分类管理，支持拖拽排序
- **分类颜色区分**: 不同级别分类用不同颜色标识，便于视觉区分

### 📊 数据可视化
- **资产趋势图**: 基于真实交易记录的历史资产变化趋势
- **资产分布饼图**: 支持不同层级的资产分布展示
- **自定义图表**: 多种图表类型，支持时间范围筛选
- **Zoom功能**: 支持图表缩放和平移操作

### 📝 账单管理
- **多币种账单**: 支持不同币种的交易记录
- **智能分类**: 自动关联到相应的资产分类
- **用途分类**: 详细的支出用途分析
- **时间精度**: 支持精确到分钟的时间记录

### 🔧 高级功能
- **拖拽排序**: 资产分类支持拖拽重新排序
- **自动记账**: 设置周期性自动记账规则
- **历史资产分析**: 反算任意时间点的资产状况
- **数据导入导出**: Excel和JSON格式的数据导入导出

## 🔧 使用说明

### 启动项目
1. 进入项目目录
2. 启动HTTP服务器：
   ```bash
   python3 -m http.server 8000
   ```
3. 浏览器访问: `http://localhost:8000`

### 基本操作
1. **添加分类**: 点击"添加分类"按钮，设置分类名称、货币类型等
2. **记录账单**: 点击"添加账单"按钮，选择分类、金额、货币类型等
3. **查看趋势**: 在资产趋势图中选择时间范围，查看资产变化
4. **拖拽排序**: 直接拖拽分类项目进行重新排序

### 高级功能
- **汇率设置**: 在设置页面自定义不同货币的汇率
- **历史分析**: 在设置页面使用"计算历史资产状况"功能
- **数据导出**: 在导入导出页面下载Excel或JSON格式数据
